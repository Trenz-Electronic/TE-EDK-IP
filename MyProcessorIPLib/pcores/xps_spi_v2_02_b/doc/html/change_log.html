Changes in v2.01.a, introduced in 11.1

11.1 - Changes in VHDL/Verilog/Netlist sources (.vhd, .v, .ngc, .edn) 


New Features:
  • Updated the slave mode functionality of the XPS SPI core.
  • Updated the master mode functionality in C_SCK_RATIO = 2 mode.
  • Updated the data registering mechanism for master mode of XPS SPI core.
Resolved Issues:
  • Resolved the issue of XPS SPI core not working in slave mode. CR: 498916 is fixed.
    • Versions that have this issue: v2.00.b
  • Resolved the issue of XPS SPI core not working in C_SCK_RATIO = 2.CR: 478338 is fixed.
    • Versions that have this issue: v2.00.b
  • Resolved the issue of asynchronous flop. CR: 481243 is fixed.
    • Versions that have this issue: v2.00.b
Known Issues / Limitations:
  • None

11.1 - Changes in tool interface files (.mpd)

  • Updated the MPD file with the actual values of C_SCK_RATIO in place of providing the range.
  • Added preferred support for qrvirtex4.
  • Added preferred support for qvirtex4.

11.1 - Changes in documentation associated with core

  • Added description for slave mode operation of the core. Added slave mode SPI register flow description.
